# CradosDex - L'encyclopédie des Crados

Vous vous rappelez des Crados?

Voici un projet de découverte du deb web avec les Crados comme sujet.

## Backend

### Lancement du serveur backend

```bash
# Dans le répertoire BACKEND
cd REPERTOIRE_DU_PROJET/BACKEND

# Récupération des dépendances
npm install

# Lancement avec auto refresh si modification des fichiers
node --watch server_cradosdex.js
```
